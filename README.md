Description:
========
This is Microsoft Minesweeper Game for Web
How to use:
========
<pre>minesweeperGame(w,h,n);</pre>
<b>w</b> -> width 
<b>h</b> -> height
<b>n</b> -> number of mines

Examples of usage:
========
<pre>minesweeperGame(9,9,10);</pre>
